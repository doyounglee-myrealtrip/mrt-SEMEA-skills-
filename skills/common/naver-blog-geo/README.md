# 📝 naver-blog-geo

> GEO/AEO 최적화 네이버 블로그 여행 글 자동 작성 스킬

---

## 이게 뭔가요?

여행지를 입력하면 **AI 검색에 잘 걸리는 네이버 블로그 글**을 자동으로 써주는 스킬이에요.

실시간 여행자 후기 리서치부터 무료 이미지 검색, 마이리얼트립 상품 링크 녹이기까지 한 번에 처리해줘요.

---

## SEO? AEO? GEO? 쉽게 설명해드릴게요

지금 검색 환경이 빠르게 바뀌고 있어요. 사람들이 네이버나 구글 검색창 대신 **ChatGPT, 클로바X, Perplexity** 같은 AI에게 직접 물어보기 시작했거든요.

| 용어 | 한 줄 설명 | 예시 |
|------|-----------|------|
| **SEO** (검색엔진 최적화) | 네이버·구글 검색 결과 상위에 뜨게 만드는 것 | "마요르카 여행" 검색 시 내 블로그가 1페이지에 노출 |
| **AEO** (답변엔진 최적화) | AI가 질문에 답할 때 내 글에서 답을 뽑아가게 만드는 것 | "마요르카 렌트카 필요해?" → AI가 내 글 인용 |
| **GEO** (생성형AI 최적화) | ChatGPT·클로바X 같은 AI가 내 글을 신뢰할 수 있는 출처로 인식하게 만드는 것 | AI가 마요르카 추천 글 생성 시 내 블로그 링크 포함 |

### 왜 지금 중요하냐면

구글 리서치에 따르면 **Z세대의 40%가 이미 검색 대신 AI에게 먼저 물어본다**고 해요. 네이버도 AI 브리핑, 클로바X 검색을 빠르게 확대하고 있어요.

SEO만 잘 해서는 부족한 시대가 됐고, **AEO + GEO까지 함께 챙겨야** 트래픽을 유지할 수 있어요.

## 이 스킬을 쓰면 어떤 효과가 있어요?

### 1. AI 검색 노출 확률 상승
각 문단을 AI가 "뽑아가기 좋은" 구조로 작성해요. 문단 하나가 독립적인 미니 답변처럼 쓰여서, ChatGPT나 클로바X가 관련 질문에 답할 때 인용할 확률이 높아져요.

### 2. 네이버 블로그 체류시간 증가
핵심 요약 박스, 비교 표, 이미지, FAQ 구조를 자동으로 넣어줘서 독자가 글을 끝까지 읽을 확률이 높아져요. 체류시간이 길수록 네이버 VIEW 탭 노출도 올라가요.

### 3. 글 작성 시간 대폭 단축
리서치 → 구조 잡기 → 초안 작성 → 이미지 찾기까지 보통 3~5시간 걸리는 작업을 **20~30분**으로 줄여줘요.

### 4. 마이리얼트립 상품 전환율 향상
상품 링크를 광고처럼 따로 넣는 게 아니라, 글의 흐름 안에 자연스럽게 녹여요. "이런 분들에게 추천" 형식으로 삽입되기 때문에 독자가 거부감 없이 클릭하게 돼요.

### 5. 실제 데이터 기반 신뢰도
AI가 한국어·영어 여행자 후기를 직접 리서치해서 실제 경험담과 수치를 글에 반영해요. 독자 신뢰도가 높아지고, AI도 신뢰할 수 있는 출처로 인식해요.

## 사용 방법

### 기본 사용

```
/naver-blog-geo [여행 주제]
```

**예시:**
```
/naver-blog-geo 마요르카 5박 6일 여행
/naver-blog-geo 오사카 3박 4일 먹방 여행
/naver-blog-geo 바르셀로나 4박 5일 자유여행
/naver-blog-geo 제주도 2박 3일 렌트카 없이
```

### 실행 흐름

```
1. 스킬 입력
   └─ /naver-blog-geo 마요르카 5박 6일

2. 자동 리서치 (약 1~2분)
   ├─ 한국어 여행자 후기 검색
   ├─ 영어 여행 팁 검색
   └─ Unsplash/Pexels 이미지 검색

3. 상품 링크 질문
   └─ "녹여드릴 마이리얼트립 상품 링크 있으신가요?"
      ├─ 있으면 → URL 붙여넣기 (여러 개 가능)
      └─ 없으면 → "없음" 입력

4. 추가 정보 질문 (한 번에)
   ├─ 타깃 독자
   ├─ 핵심 메시지
   └─ 직접 경험 여부

5. 완성 글 출력
   ├─ GEO/AEO 최적화 본문
   └─ 이미지 삽입 가이드 (다운로드 링크 + alt text)
```

### 상품 링크 활용 예시

```
/naver-blog-geo 마요르카 5박 6일 여행

→ 리서치 완료 후 Claude가 질문:
  "상품 링크 있으신가요?"

→ 아래처럼 링크 붙여넣기:
  https://experiences.myrealtrip.com/products/3851372
  https://experiences.myrealtrip.com/products/5868228
  https://experiences.myrealtrip.com/products/4974572
```

## 결과물 구성

스킬 실행 후 아래 내용이 출력돼요.

```
[블로그 본문]
├─ 제목 (구체적 상황 + 연도)
├─ 도입부
├─ 💡 핵심 요약 박스      ← AI featured snippet 최적화
├─ 본문 (H2 구조, 이모지 소제목)
├─ 비교 표 (A vs B)       ← "어느 게 나아요?" 질문 대응
├─ 실제 여행자 경험 섹션  ← 리서치 기반
├─ FAQ (롱테일 질문)      ← AI 검색 질문 매칭
└─ 마무리

[이미지 삽입 가이드]
└─ 섹션별 이미지 다운로드 링크 + 권장 alt text 표
```

---

## 이미지 업로드 방법

스킬이 Unsplash/Pexels 이미지 링크를 제공해요. 네이버 블로그는 외부 URL 직접 삽입이 안 되기 때문에 아래 순서로 올려주세요.

```
1. 제공된 URL 클릭 → 이미지 다운로드
2. 네이버 블로그 편집창 → 이미지 업로드
3. 권장 alt text를 사진 설명 칸에 입력
```

모든 Unsplash·Pexels 이미지는 **상업적 무료 사용 가능**해요.

## 자주 묻는 질문

**Q. 여행 외 주제에도 쓸 수 있나요?**
현재는 여행 콘텐츠에 최적화되어 있어요. 다른 카테고리(맛집, 숙소 단독 리뷰 등)는 추후 별도 스킬로 추가 예정이에요.

**Q. 직접 경험이 없어도 괜찮나요?**
괜찮아요. 실제 여행자 후기 리서치를 자동으로 해서 내용을 채워줘요. 직접 경험이 있다면 입력했을 때 글의 신뢰도가 더 높아져요.

**Q. 결과물을 바로 올려도 되나요?**
초안으로 활용하고, 본인 경험이나 최신 정보를 추가로 검수한 후 올리는 걸 권장해요. AI가 리서치한 내용이라 간혹 최신 현지 정보와 다를 수 있어요.

**Q. 상품 링크는 몇 개까지 넣을 수 있나요?**
제한 없어요. 여러 개 넣으면 각각의 내용을 읽고 모두 자연스럽게 녹여줘요.
