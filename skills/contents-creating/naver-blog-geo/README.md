# 📝 naver-blog-geo

> GEO/AEO 최적화 네이버 블로그 여행 글 자동 작성 스킬

---

## 소개

여행지를 입력하면 **AI 검색에 최적화된 네이버 블로그 글**을 자동으로 작성합니다.

실시간 여행자 후기 리서치, 무료 이미지 검색, 마이리얼트립 상품 링크 통합까지 한 번에 처리합니다.

---

## 배경: SEO에서 AEO·GEO로

검색 환경이 변화하고 있습니다. 사용자들이 네이버·구글 검색창 대신 **ChatGPT, 클로바X, Perplexity** 같은 AI에 직접 질문하는 비중이 빠르게 늘고 있습니다.

- **SEO** (검색엔진 최적화) — 네이버·구글 검색 결과 상위 노출. 예: "마요르카 여행" 검색 시 블로그 1페이지 노출
- **AEO** (답변엔진 최적화) — AI가 질문에 답할 때 내 콘텐츠를 인용하도록 최적화. 예: "마요르카 렌트카 필요해?" → AI가 내 글 인용
- **GEO** (생성형AI 최적화) — AI가 콘텐츠를 신뢰할 수 있는 정보 출처로 인식하도록 최적화. 예: AI가 마요르카 추천 글 생성 시 내 블로그 링크 포함

구글 리서치에 따르면 **Z세대의 40%가 검색 대신 AI를 먼저 활용**합니다. 네이버 역시 AI 브리핑과 클로바X 검색을 빠르게 확장하고 있습니다. SEO만으로는 트래픽을 유지하기 어렵고, **AEO·GEO를 함께 고려한 콘텐츠 전략**이 필요합니다.

## 기대 효과

### 1. AI 검색 노출 확률 상승
각 문단을 AI가 독립적으로 추출할 수 있는 구조로 작성합니다. ChatGPT·클로바X가 관련 질문에 답변할 때 해당 문단을 인용할 확률이 높아집니다.

### 2. 네이버 블로그 체류시간 증가
핵심 요약 박스, 비교 표, 이미지, FAQ 구조를 자동으로 포함해 독자의 완독률을 높입니다. 체류시간이 길수록 네이버 VIEW 탭 노출에 유리합니다.

### 3. 콘텐츠 제작 시간 단축
리서치 → 구조 설계 → 초안 작성 → 이미지 수집까지 보통 3-5시간 걸리는 작업을 **10-15분**으로 단축합니다.

### 4. 마이리얼트립 상품 전환율 향상
상품 링크를 독립된 광고 형태가 아닌 본문 흐름 안에 자연스럽게 통합합니다. 맥락에 맞는 삽입 방식으로 클릭 전환율을 높입니다.

### 5. 실제 데이터 기반 콘텐츠 신뢰도
한국어·영어 여행자 후기를 실시간으로 리서치해 구체적인 수치와 경험담을 반영합니다. 독자 신뢰도와 AI 인용 가능성을 동시에 높입니다.

### 6. 마이리얼트립 브랜드 자연 노출
"마이리얼트립"을 별도로 지정하지 않아도 본문에 최소 2회 이상 자연스럽게 녹입니다. 예약 방법·가격·특징 등 맥락과 함께 언급해 광고성 문구 없이 브랜드를 노출합니다.

## 사용 방법

### 기본 사용

```
/naver-blog-geo [여행 주제]
```

**예시:**
```
/naver-blog-geo 마요르카 5박 6일 여행
/naver-blog-geo 오사카 3박 4일 먹방 여행
/naver-blog-geo 바르셀로나 4박 5일 자유여행
/naver-blog-geo 제주도 2박 3일 렌트카 없이
```

### 실행 흐름

```
1. 스킬 입력
   └─ /naver-blog-geo 마요르카 5박 6일

2. 사전 질문 (한 번에)
   ├─ 마이리얼트립 상품 링크 여부 (선택, 여러 개 가능)
   └─ 이미지 검색 여부 (예 / 아니오)
      ※ "아니오" 선택 시 훨씬 빠르게 완료됩니다.

3. 여행자 후기 자동 리서치
   ├─ 한국어 후기 검색
   └─ 영어 여행 팁 검색

4. 추가 정보 입력 (한 번에)
   ├─ 타깃 독자
   ├─ 핵심 메시지
   ├─ 직접 경험 여부
   └─ SEO 키워드 (선택)

5. 블로그 본문 출력
   └─ GEO/AEO 최적화 완성 글

6. 이미지 가이드 출력 (이미지 "예" 선택 시에만)
   └─ 섹션별 이미지 다운로드 링크 + alt text 표
```

### 상품 링크 활용 예시

```
/naver-blog-geo 마요르카 5박 6일 여행

→ 리서치 완료 후 상품 링크 입력:
  https://experiences.myrealtrip.com/products/3851372
  https://experiences.myrealtrip.com/products/5868228
  https://experiences.myrealtrip.com/products/4974572
```

## 결과물 구성

```
[블로그 본문]
├─ 제목 (구체적 상황 + 연도)
├─ 도입부
├─ 💡 핵심 요약 박스      ← AI featured snippet 최적화
├─ 본문 (H2 구조, 이모지 소제목)
├─ 비교 표 (A vs B)       ← "어느 게 나아요?" 질문 대응
├─ 실제 여행자 경험 섹션  ← 리서치 기반
├─ FAQ (롱테일 질문)      ← AI 검색 질문 매칭
└─ 마무리

[이미지 삽입 가이드]
└─ 섹션별 이미지 다운로드 링크 + 권장 alt text 표
```

## 이미지 업로드

스킬이 제공하는 Unsplash·Pexels 이미지는 **상업적 무료 사용**이 가능합니다. 네이버 블로그는 외부 URL 직접 삽입이 지원되지 않으므로 아래 순서로 진행하세요.

```
1. 제공된 URL로 이미지 다운로드
2. 네이버 블로그 편집창에서 이미지 업로드
3. 권장 alt text를 사진 설명 칸에 입력
```

## FAQ

**Q. 여행 외 주제에도 사용할 수 있나요?**
현재는 여행 콘텐츠에 최적화되어 있습니다. 맛집, 숙소 단독 리뷰 등 다른 카테고리는 별도 스킬로 추가할 예정입니다.

**Q. 직접 경험이 없어도 사용 가능한가요?**
가능합니다. 실제 여행자 후기 리서치를 자동으로 수행해 콘텐츠를 구성합니다. 직접 경험을 입력하면 E-E-A-T 신뢰도 측면에서 더 유리합니다.

**Q. 결과물을 바로 발행해도 되나요?**
초안으로 활용하되, 최신 현지 정보와 본인 경험을 추가 검수한 후 발행하는 것을 권장합니다.

**Q. E-E-A-T가 뭔가요?**
Google이 콘텐츠 품질을 평가하는 기준입니다 (Experience·Expertise·Authoritativeness·Trustworthiness). AI도 동일한 기준으로 신뢰도를 판단해요.

직접 여행 경험이 있다면 고지문처럼 따로 쓰지 말고, 본문 안에 자연스럽게 녹이세요.
> ❌ "이 글은 직접 여행한 경험을 바탕으로 작성했습니다" (인위적)
> ✅ "저도 10시쯤 갔다가 줄 서느라 1시간 날렸거든요" (자연스러운 경험담)

직접 경험이 없으면 이 항목은 건너뜁니다.

**Q. SEO 키워드는 몇 개까지 지정할 수 있나요?**
제한은 없지만 3-5개를 권장합니다. 너무 많으면 억지로 끼워넣는 느낌이 생겨 GEO/AEO에 역효과가 날 수 있습니다.

**Q. 상품 링크는 몇 개까지 입력할 수 있나요?**
제한 없습니다. 여러 개 입력 시 각 상품 내용을 읽고 본문에 자연스럽게 통합합니다.
